## 概要

プログラミング学習を開始して 1 ヶ月ほど経過した後に、学習したことのアウトプットとして Web サイト作成しました。

- レスポンシブデザイン（スマホのみ）
- メニューリストの画像スライダーはプラグインを使用せず、jQurty を使用し自分で作成
- 画像スライダーは**this の向き先を意識し、クロージャーを使用**して作成

## 使用技術

- CSS
- jQuery（Ver 3.4.1）
